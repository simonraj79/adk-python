# GCP IAM Connector Auth

Manages complete lifecycle of access token using the GCP IAM Connector
Credential service.

## Setup

1.  **Install Dependencies:**
    ```bash
    pip install "google-adk[iam_connectors]"
    ```

2.  **Configure:**
    Specify the IAM Connector configuration like Connector name, scopes etc
    using the `GcpIamConnectorAuth` as `auth_scheme` while configuring
    `McpToolset` Refer to this link for detailed steps (TODO)
